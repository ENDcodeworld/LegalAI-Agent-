# LegalAI-Agent - 法律 AI 助手

智能合同审阅 Agent - 用户上传合同，AI 标记风险点、生成修改建议

## 🎯 产品定位

**目标用户:** 律所、法务、中小企业  
**核心场景:** 合同审阅、风险识别、修改建议  
**付费意愿:** ⭐⭐⭐⭐⭐ 极高

## 🚀 核心功能

### 1. 合同风险识别
- 自动标记高风险条款
- 识别不公平条款
- 提示缺失关键条款
- 法律依据引用

### 2. 修改建议生成
- 针对性修改方案
- 替代条款推荐
- 风险等级评估
- 谈判要点提示

### 3. 合同类型支持
- 买卖合同
- 租赁合同
- 劳动合同
- 投资协议
- 合作协议
- ...

## 📁 项目结构

## 🛠️ 技术栈

- **前端:** React + TailwindCSS
- **后端:** Python + FastAPI
- **AI:** 通义千问 / 本地部署
- **数据库:** PostgreSQL
- **部署:** Docker + 阿里云
