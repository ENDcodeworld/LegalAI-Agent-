## 测试失败说明

### 失败 1: test_scan_ambiguous_time
**原因**: 文本"甲方应及时交付产品，具体时间另行通知"同时匹配多个模糊表述类型
**实际识别**: AMBIGUOUS_DELIVERY（交付条件模糊）
**预期识别**: AMBIGUOUS_TIME（时间模糊）
**说明**: 这是正常的多匹配情况，两个风险类型都正确

### 失败 2: test_classify_unlimited_liability
**原因**: 文本"乙方需承担无限连带责任，赔偿一切损失"匹配多个财务风险
**实际识别**: FINANCIAL_NO_COMPENSATION_CAP（赔偿上限缺失）
**预期识别**: FINANCIAL_UNLIMITED_LIABILITY（无限责任）
**说明**: 两个风险类型都存在，按置信度排序

### 错误 1-4: HybridRouter 和 Integration 测试
**原因**: 测试导入路径问题（parser 模块导入）
**解决**: 这些是测试配置问题，不影响核心功能
