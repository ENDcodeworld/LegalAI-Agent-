## ⚙️ 五、自动发货设置教程

### 方法一：闲鱼自动发货（推荐）

#### 5.1 开通自动发货
1. 进入 **"我的"** → **"创作中心"**
2. 找到 **"自动发货"** 功能
3. 申请开通（可能需要实名认证）
4. 设置自动回复内容

#### 5.2 配置自动回复
```

您的 108 份合同模板已准备好：

📦 下载地址：[百度网盘链接]
🔑 提取码：[4 位提取码]

💡 温馨提示：
1. 全部为 Word 格式，可直接编辑
2. 包含填写指南和注意事项
3. 如有问题请留言，我会尽快回复

祝您使用愉快！✨
```

#### 5.3 上传网盘文件
1. 将 108 份合同模板打包成 ZIP 文件
2. 上传到百度网盘
3. 创建分享链接（永久有效）
4. 设置提取码
5. 将链接和提取码填入自动回复
